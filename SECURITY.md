# Security Policy

## Supported Versions

The following versions of `picotoml` currently receive security updates:

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you've discovered a security vulnerability in `picotoml`, please report it by sending an e-mail to [INSERT EMAIL ADDRESS].

Please include:
* The steps required to reproduce the vulnerability.
* Any potential impact or risk associated with the vulnerability.
* Your name or handle for credit (optional).

Do not disclose security vulnerabilities publicly until they have been resolved. We will investigate and respond to all reports as quickly as possible.
